EzLab

When set up virtual machine or even bigger like a hacking lab for the first time, I took so much time reading documentation and clicking buttons, and even later when being familiar with the concept, the later one still happens to me. That's why I came up with this project. It is designed to help Windows user set up their virtual machine and hacking lab faster nad more convenient. At the mean time, it can only work on Virtual Box running on Windows, but I'm working on some changes to make it work on other machine like Linux or Mac and also VMware.
